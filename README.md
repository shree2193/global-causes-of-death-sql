# Global Causes of Death — SQL Analysis

Analyzed 200,000+ rows of WHO mortality data using PostgreSQL.

## Dataset
Source: Our World in Data / Global Burden of Disease
- 201,762 rows across 30+ causes of death
- 180+ countries, 1990–2019

## Skills demonstrated
- Table design and schema creation
- CSV data import and validation
- GROUP BY aggregations
- Window functions (LAG, RANK)
- Views and indexes
