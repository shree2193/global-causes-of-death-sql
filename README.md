# Global Causes of Death — SQL & Tableau Analysis

## Overview
Built a PostgreSQL database analyzing 200,000+ rows of global mortality 
data across 180+ countries and 30+ causes of death (1990–2019). Wrote 
10 SQL queries and built an interactive Tableau dashboard.

## Live Dashboard
https://public.tableau.com/app/profile/shree.m1081/viz/GlobalCausesofDeathAnalysi/Dashboard1

## Tools
PostgreSQL | pgAdmin | Tableau Public

## Dataset
- Source: Our World in Data / Global Burden of Disease
- 201,762 rows across 30+ causes of death
- 180+ countries, 1990–2019

## SQL Skills Demonstrated
- Window functions — LAG, RANK, rolling averages
- CTEs (WITH clause)
- GROUP BY aggregations and FILTER
- Views and stored functions
- Indexing for query optimization

## Dashboard Charts
- Top 10 global causes of death (2019)
- Cardiovascular disease trend 1990–2019
- Year over year % change in deaths
- Countries above average for diabetes and cardiovascular deaths

## Resume Summary
Designed and built a relational PostgreSQL database with 200K+ records 
across 30+ causes of death. Authored 10 analytical SQL queries using 
window functions (LAG, RANK, rolling averages), CTEs, and aggregation 
filters. Built an interactive 4-chart Tableau Public dashboard. Applied 
composite indexing and created reusable views and stored functions.
